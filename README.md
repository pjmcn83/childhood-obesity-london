# Environmental Predictors of Child Obesity in London
## A Supervised Machine Learning Analysis at MSOA Level

This repository contains the final submission for the CASA0006 Data Science for Spatial Systems module, UCL MSc Urban Spatial Science.

**Author:** Paul McNicholas

### Notebooks

- `child-obesity-london-main.ipynb` — main analysis notebook. Can be run directly using the preprocessed dataset in `data/processed/`.

- `child-obesity-london-prep.ipynb` — preprocessing notebook. Processes data from the original sources. Some datasets are stored in `data/raw/` for convenience; others connect directly to APIs or online datastores — full details are in the notebook.

### Data

| File | Description |
|---|---|
| `data/processed/london_obesity_master_dataset.csv` | Clean master dataset, ready for analysis |
| `data/raw/` | Raw source files where applicable |
